Where to start

A simple example

$$ d_{1} + d_{2} + d_{3} + d_{4} + d_{5} + ... + d_{k-1} + d_{k} $$

  • single cpu 2.0GHz | 0.5 ns | 4 pipelines
  • + | C.float.add | 5 cycle
  • k | 2*10^{8} adds | 1s
  • datasize | 1TB | simple add | 12min

Case1

$$ d_{1} + d_{2} + d_{3} + d_{4} + d_{5} + d_{6} ... + d_{k-1} + d_{k} $$

  • d | float array
  • + | user-defined func | 2000 cycles
  • k | 0.5*10^{7} +s
  • datasize | 1TB | user-defined func | 8.5h

Case2

$$ d_{1} + d_{2} + d_{3} + d_{4} + d_{5} + ... + d_{k-1} + d_{k} $$

  • $$+ | \cap or \cup $$
  • bang

Optimization always confuse

  • Optimization != distributed computing

    • $$Efficiency \longrightarrow qualitative(Case1)$$

    • $$Distributed memory \longrightarrow qualitative(Case2)$$


  • Distributed/Parallel computing make things work
